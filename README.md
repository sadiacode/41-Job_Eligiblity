# Job Eligiblity Check
## Description
This project is a simple python program that demonstrates the if-else statement in Python. The if-else statement is used to make decisions by executing one block of code when a condition is **True** and another block when the condition is  **False**.
## Syntax
```python
if condition:
   # Code to execute if the condition is True
else:
   # Code to execute if the condition is False
```
## Technologies Used
- Pycharm
- Python 3
## How to run
1. Clone the repository.
2. Open the project in pycharm.
3. Run the 'main.py' file.
4. View the output.
## Sample Output
```text
Enter your age: 34
Do you have any degree?(yes/no): yes

Eligible for the job.
```
